Fillwave graphics engine instalation.

Requirements

  • PC with OpenGL 3.3+ and GLSL 330+ support.
  • Android with OpenGL 3.0+, GLSL ES 300+ support. (API > 18, Rev. > 4.3)

Ubuntu (Debian):

1. Install all dependencies:


sudo apt-get install libfreetype6 libfreetype6-dev libassimp3 libassimp-dev libglm-dev libglm libx11-dev libglu-dev

2.a) Install from the package directly ...

  • http://filipwasil.bitbucket.org/downloads/libfillwave-1.9.7-Linux.deb
  • http://filipwasil.bitbucket.org/downloads/libfillwave-dev-1.9.7-Linux.deb

2.b) ... or add fillwave to software sources:


deb http://filipwasil.bitbucket.org/downloads ./
#deb-src http://filipwasil.bitbucket.org/downloads ./

3. Update and install:

sudo apt-get update

sudo apt-get install sudo apt-get install libfillwave libfillwave-dev fillwave-examples

Fedora (Red hat):

1. Install all dependencies:


sudo yum install glm-devel.x86_64 glm.x86_64 assimp.x86_64 assimp-devel.x86_64 glew.x86_64 glew-devel.x86_64 freetype-devel.x86_64 freetype.x86_64

2. Copy fillwave.repo file in Your /etc/yum.repos.d/, update and install.

sudo yum update

sudo yum install sudo apt-get install libfillwave libfillwave-dev fillwave-examples

GLFW3 for context creation on PC

The easiest way to start playing with Fillwave engine on PC is to create context with GLFW3 and use examples provided. GLFW3 is in default repositries under Ubuntu 14.10+. In case of older system just add following lines to /etc/apt/sources.list:


deb http://ppa.launchpad.net/keithw/glfw3/ubuntu trusty main
deb-src http://ppa.launchpad.net/keithw/glfw3/ubuntu trusty main

GLFW3 installation


sudo apt-get install libglfw3
